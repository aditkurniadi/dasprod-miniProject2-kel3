# Mini Project 2: Analisis Data Kasus HIV/AIDS di Aceh (2018-2023)

Proyek ini merupakan bagian dari tugas Mini Project 2 yang bertujuan untuk menganalisis dataset "Jumlah Kasus HIV/AIDS Menurut Kelompok Umur di Provinsi Aceh" dari tahun 2018-2023.


**Sumber Data:**
* **Dataset:** [Jumlah Kasus HIV/AIDS Menurut Kelompok Umur di Provinsi Aceh](https://data.acehprov.go.id/dataset/b53d4999-0cb8-4fe7-9d64-3218976feeb2/resource/8e76cafc-6b24-4d24-b839-1d01b5035158/download/jumlah-kasus-hiv-aids-menurut-kelompok-umur-di-aceh.csv)

## Pembagian Peran

Proyek ini dibagi menjadi 4 direktori utama yang merepresentasikan setiap peran dalam tim:

1. Data Engineer
    Melakukan data cleansing dan mengolah data mentah yang tidak konsisten menjadi data bersih yang siap dianalisis.

2.  Data Analyst
     Mengolah data bersih menjadi tabel-tabel ringkasan yang spesifik dan siap untuk divisualisasikan.

3.  Data Visualizer
    Mengkonversi data dari Data Analyst menjadi visualisasi grafis menggunakan Matplotlib.

4.  Presenter
    Merangkai seluruh hasil kerja (data bersih, visualisasi) menjadi presentasi yang mudah dipahami.
